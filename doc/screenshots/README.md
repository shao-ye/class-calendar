# 界面截图目录

把项目运行后的真实界面截图放在本目录，根 `README.md` 的「界面预览」会自动引用。

## 需要的文件

| 文件名 | 对应界面 | 建议内容 |
| --- | --- | --- |
| `calendar.png` | 日历视图 | 月视图，能看到彩色课程标签、计划课虚线、假日标记 |
| `stats.png` | 统计分析 | 统计页，含课时/花费柱状图 |
| `courses.png` | 课程库 | 课程列表，含课时进度条 |
| `record.png` | 当日记录 | 点开某天的记录抽屉或编辑表单 |

## 建议规范

- 手机端尺寸截图（产品为移动端优先），宽度建议 750~1080px
- 使用演示数据，**不要包含真实孩子姓名、照片或隐私信息**
- 格式优先 `png`，单张控制在 500KB 以内（可适当压缩）
